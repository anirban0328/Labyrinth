ReadMe

- The application is build on Java.
- Main starting point of the application is “MainProcessor.java”.
- User needs to enter FileName (without appending .txt) which has the input data.
- After the labyrinth is displayed, user can enter the moves.
- User can exit the application by either giving empty moves or by typing exit (case insensitive).
- If invalid user data is passed then application will exit with invalid user input exception.
